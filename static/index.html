Проект выполняет сегментацию покупателей торгового центра с помощью кластеризации KMeans, чтобы разделить клиентов на группы по покупательскому поведению.

Что такое Unsupervised Learning
Unsupervised Learning — это обучение без целевых меток: модель сама находит скрытую структуру в данных. В этом проекте алгоритм KMeans формирует кластеры клиентов по признакам дохода и индекса трат.
